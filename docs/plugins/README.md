# Plugins Guide

Siren supports extensibility. Add custom agents, attacks, datasets, model providers, and sandbox managers by following the established patterns and registering them with the component registry.

For an overview of the plugin system, see [Plugin System](./plugin_system.md).

## Component Types

- [**Agents**](./agents.md) - Implement custom agent logic for task execution
- [**Attacks**](./attacks.md) - Create custom attack strategies for injection testing
- [**Datasets**](./datasets.md) - Define custom task datasets for evaluation
- [**Environments**](./environments.md) - Build custom execution environments for tasks
- [**Model Providers**](./model_providers.md) - Integrate custom LLM providers
- [**Sandbox Managers**](./sandbox_managers.md) - Implement custom sandbox isolation strategies
